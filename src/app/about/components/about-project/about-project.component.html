In short

MEAN, ngrx and nebular based quiz platform. Made by developer for developers. Open source.

Motivation

There are existing quiz platforms, but either they are not focused on developer specific needs (like code highlighting), or some nice-to-have features are missing or not configurable.

Features

Some features:
  • Explanation of answers
  • Comments on answers
  • Sharing result with friends
  • Propose your own question or answer
  • Create your own quiz
